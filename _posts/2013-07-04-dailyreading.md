[Sizing (Web) components](https://medium.com/front-end-development/8f433689736f)  
By using __EM__ unit, we can easily change component's size when its font size change.

```css
.Component {
  font-size: 16px;
  
  width: 5em;
  height: 2em;
  border-radius: .5em;
  border: 1px solid gold;
}
```
